# Patient Simulator Bot

A voice bot that tests Pretty Good AI's healthcare agent by simulating patient calls using Twilio, OpenAI Whisper, ChatGPT, and ElevenLabs.

## 🚀 Quick Start

### Prerequisites
- Python 3.9+
- Twilio account with phone number
- OpenAI API key (for Whisper and ChatGPT)
- ElevenLabs API key
- ngrok (for local development)

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/patient-simulator-bot
cd patient-simulator-bot
